# see-quil

A repository containing "Mary Shelley's Frankenstein" and a simple Python tool to read it.

## Prerequisites

To run the script in this repository, you'll need to have Git and Python installed on your system.

### Installing Git

Git can be installed using Homebrew:
```bash
brew install git
```


### Installing Python

Python 3 is often pre-installed on macOS, but you can install the latest version via Homebrew:
```bash
brew install python
```

## How to Run

1.  **Clone the Repository**:
    ```bash
    git clone <repository-url>
    cd see-quil
    ```
2.  **Run the Script**:
    ```bash
    python3 main.py
    ```
    This will print out the entire text of Frankenstein line by line.

## Data
The text for "Mary Shelley's Frankenstein" is located in `data/frankenstein.txt`. This text is in the public domain, sourced from Project Gutenberg.
